# Frith AI Platform

Legal AI tools platform built with Next.js, Prisma, and modern AI models.

## Repository Structure

```
frith/
├── dev/          # Development source code and configuration
│   ├── app/      # Next.js application code
│   ├── components/ # React components
│   ├── lib/      # Utility libraries
│   ├── prisma/   # Database schema and migrations
│   └── ...       # All project source files
│
├── prod/         # Production deployment scripts and guides
│   ├── *.sh      # Deployment scripts
│   └── *.md      # Deployment documentation
│
└── notes/        # Project documentation and planning
    └── *.md      # Development notes, phase completions, specs
```

## Quick Start

1. Navigate to the `dev/` folder for all development work
2. See `prod/` for deployment guides
3. Check `notes/` for project documentation

## Development

```bash
cd dev/
npm install
npm run dev
```

For complete setup instructions, see `prod/DEPLOYMENT-GUIDE.md`
